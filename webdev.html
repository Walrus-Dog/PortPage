Detroix

Detroix was our first project for the Mississppi Coding Academy. We started with it by making 3 versions, only using either position, float, or flex. I honestly had a very hard time initially with this project. The only positional tool I really understood was flex, and even then it wasn't a very good understanding. The position and float really made me question if i wanted to keep doing this, it was a rough week but I finished it all somehow. We then went back to this project and built it in bootstrap, which is where it links to on GitHub. Detroixは、Mississppi Coding Academyの最初のプロジェクトでした。まず、position、float、flexのいずれかだけを使って3つのバージョンを作りました。正直なところ、最初はこのプロジェクトにとても苦労しました。位置決めツールで本当に理解できたのはflexだけで、それもあまりよく理解できていませんでした。ポジションとフロートについては、本当にこの仕事を続けていいのか疑問に思いました。その後、このプロジェクトに戻り、GitHubのリンク先であるbootstrapでビルドしました。

Bloom

Bloom was our second project for MSCA. Much like Detroix, we had to make it using the 3 positioning styles. Something clicked much better with this project than with Detroix, for whatever reason I had a much easier time with this one. We didn't go back over Bloom when we learned bootstrap, so it is only in the 3 positional tools. BloomはMSCAの2番目のプロジェクトでした。Detroixと同じように、3つのポジショニング・スタイルを使って作らなければなりませんでした。このプロジェクトでは、Detroixよりも何かがうまく噛み合っていて、なぜか私はこのプロジェクトにずっと楽に取り組むことができました。bootstrapを学んだときにBloomを見返していなかったので、3つのポジショニングツールだけになっています。

Aditii

Our next project was Aditii. We were told to chose the positional tool we were most comfortable with and only use that, so I built mine only using flexbox. This project wasn't that difficult compared to Detroix, as by now I had a much better understanding of how things work. 次のプロジェクトはAditiiでした。一番使いやすい位置調整ツールを選んで、それだけを使うように言われたので、私はフレックスボックスだけを使って作りました。このプロジェクトは、Detroixに比べてそれほど難しいものではありませんでした。というのも、この頃には物事がどのように機能するのかをずっと理解していたからです。

GraphicUmbrella

Following Aditii, we were given 3 pages to build in bootstrap. The first was GraphicUmbrella. Bootstrap made this a very easy and actually enjoyable process. It was our first page to make responsive, which was a fun challenge. Aditiiに続いて、bootstrapで構築する3つのページが与えられました。最初のページは GraphicUmbrella でした。Bootstrapのおかげで、この作業はとても簡単で、実際に楽しいものでした。レスポンシブ対応のページを作るのは初めてでしたが、これは楽しいチャレンジでした。

Venedor

Venedor was next. This was a fun page to build, much like GraphicUmbrella. Not much to say about it other than how much of a help bootstrap is. 次はVenedorです。これはGraphicUmbrellaのように、作っていて楽しいページでした。ブートストラップがどれだけ役に立ったかということ以外、あまり言うことはありません。

Restaurant

Our last page before making our portfolio, Restaurant was a group code along. Simple but fun none the less. ポートフォリオを作る前の最後のページで、グループコードを使ったレストランです。シンプルですが、楽しいページです。
