Orange Blade

Orange Blade is a coop boss gauntlet concept that can work as either a ttrpg, a 3d souls like, or a 2d bullet hell/danmaku. This game will be up to 3 player co-op, with the bosses scaling appropriately based on player amount. Players have to defeat each boss in succession. Each player starts with 3 "lives", once all lives are used the ENTIRE party must reset to the first boss at the very start of the game. Each "stage" consists of two "phases". Completion of a stage rewards all players with an extra life, for a total of 9, if they don't die once during their climb of the mountain. The theme of this game is classical fantasy Japanese, the bosses are all themed off of this and the player characters will be as well. Progress on this project is on hold until I feel I have progressed enough with programming to start laying down code. The game mechanics and boss fights are all finished on paper, so it will just be a matter of making them in Unity. Please E-Mail me if interested. Orange Bladeは、協力型のボスガントレットというコンセプトで、TRPG、3Dソウルのようなもの、2DのBULLET HELL/DANMAKUのようなものとして機能します。このゲームは3人までの協力プレイが可能で、ボスはプレイヤーの人数に応じて適切にスケールアップします。プレイヤーは各ボスを連続して倒す必要があります。各プレイヤーは3つの "ライフ "を持っていますが、すべてのライフを使い果たすと、パーティ全員がゲーム開始時の最初のボスにリセットされます。各ステージは2つの「フェーズ」で構成されています。ステージをクリアすると、山登り中に一度も死ななかった場合、すべてのプレイヤーに追加のライフが与えられ、合計9つのライフを得ることができます。このゲームのテーマは日本の古典的なファンタジーで、ボスはすべてこのテーマに基づいていますし、プレイヤーのキャラクターも同様になります。このプロジェクトの進行は、コードを書き始めるのに十分なプログラミングの上達が感じられるまで保留されています。ゲームの仕組みやボス戦はすべて紙の上で完成していますので、あとはUnityで作るだけです。興味のある方は私に メール (E-Mail) をください。

MMO

This is my long term passion project, that will porbably not be finished anytime soon. It will take place in a modren day setting of my own invention, featuring many real wold locations, though it will initially based in Japan. The world this takes place in is much like our own, only every ledgend, urban myth, ghost and ghoul is real. Currently it features 9 classes, with 3 specializations between them, though this is subject to change. It will have dungeons, end game raids, and pvp. The main focus of this project currently is getting everything that is possible to get down on paper down. Classes, abilities, lore, locations, ect. This is a huge undertaking and will be something I probably work on for years. For more information about it, please send me an E-Mail. これは私の長期にわたる情熱的なプロジェクトであり、おそらくすぐには完成しないでしょう。この作品は、私が独自に考案した現代を舞台にしたもので、多くの現実世界の場所が登場しますが、最初は日本を舞台にしています。舞台となる世界は私たちの世界とよく似ていますが、すべての伝説や都市伝説、幽霊やグールは実在します。現在のところ、9つのクラスがあり、その間に3つの専門分野がありますが、これは変更される可能性があります。ダンジョン、エンドゲームレイド、PvPなどがあります。現在、このプロジェクトの主な焦点は、紙に書き出すことが可能なものをすべて書き出すことです。クラス、アビリティ、ロアー、ロケーションなどです。これは巨大な事業であり、おそらく何年もかけて取り組むものになるでしょう。このプロジェクトについての詳しい情報は、私に メール (E-mail) を送ってください。
